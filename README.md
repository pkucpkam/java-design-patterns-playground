# Java Design Patterns

A practical collection of Design Pattern implementations in Java, focused on real-world backend development scenarios.

## 🎯 Goal

This repository is created to:

* Learn and master GoF Design Patterns
* Understand when and why a pattern should be used
* Compare bad vs good implementations
* Apply patterns to realistic business problems
* Build a strong Java backend engineering portfolio

Instead of using toy examples, every pattern is demonstrated through a business-oriented use case commonly found in enterprise applications.
